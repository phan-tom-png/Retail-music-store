# Retail Music Store

A small static site / demo project: a place where soundtrack is good for lifestyle.

This repository currently contains HTML files (100% HTML). The README below gives quick info to help make the project look active and ready for GitHub Pages.

## About

Retail Music Store is a simple collection of static web pages showcasing soundtracks and playlists suitable for different lifestyles — study, workout, chill, and more.

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>Retail Music Store</title>
  </head>
  <body>
    <h1>Retail Music Store</h1>
    <p>Soundtracks for your lifestyle — study, work, relax, and move.</p>
    <p>To enable GitHub Pages, add an index.html and enable Pages in the repository settings.</p>
  </body>
